Formularios en HTML

El objetivo de los formularios en HTML es crear la comunicación o iteractividad con mi usuario final.

  • input:Se usa para crear controles interactivos con la finalidad de recibir datos del usuario.

      • Atributos

    1. type: Tipo de informacion que quieres recibir para poder interpretarla como tal. PJ. Text, Number, Password, email,etc.
    2. Placeholder: Podemos guiar al usuario escribiendo la instruccion o la informacion que tiene que poner.
    3. maxlength: Largo maximo de carateres que va a admitir el input.
    4. Required: Hace obligatorio al input al que se lo asignes, volviendo al input faltante. (Validación sencilla)
    5. name: Es para identificar los input en archivos como php.
    6. id: Sirve para darle un valor como identificador unico.
    7. for: Me da pie a que empiecen a escribir en el input seleccionado.
